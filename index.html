The labs
How it works — add a lab in three steps
01 · run

Run the head-to-head

run-lab.sh sends every task through every model via Claude Code (Max plan, no API key). Served model + tokens + cost come from billing telemetry — not the model's self-report.

02 · grade

Grade to data

grade.py turns raw runs into an auditable results.json — metrics only. The models' generated code is discarded.

03 · decide

Update the routing

Refresh routing.json and this page picks it up. The recommendation is the point; the decks are the evidence.
